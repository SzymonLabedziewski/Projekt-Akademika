# Projekt-Akademika
_Aplikacja działa wyłącznie w środowisku uczelnianym AGH. Baza danych PostgreSQL znajduje się na serwerze pascal.fis.agh.edu.pl i nie jest dostępna publicznie. Uruchomienie projektu wymaga aktywnego konta uczelnianego oraz połączenia z siecią AGH (bezpośrednio lub poprzez tunel SSH). Backend aplikacji (Node.js) łączy się z bazą danych przez tunelowanie portów, a interfejs użytkownika działa lokalnie w przeglądarce._

_Katalog node_modules nie jest częścią projektu i nie został dołączony do archiwum. Wymagane biblioteki instalowane są lokalnie poprzez npm install na podstawie pliku package.json._
